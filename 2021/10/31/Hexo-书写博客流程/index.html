书写博客

这里选择的 Markdown 编辑器是 VSCode，所以，我们来到 VSCode，这里建议使用 Ctrl + P 快捷键进行快速搜索刚才创建的博客，

然后，我们就可以对文章进行任意地编写了，注意，这里在 VSCode 中使用了 Markdown Previewed Enhanced 插件，即，将左侧的文本实时渲染成相应的效果在右侧边栏进行显示，如下图所示：

然后，简单解释一下博客页首的一些配置性信息：

  • title 是博客的标题，这个在使用 hexo new '' 创建文章时会自动生成；
  • date 是创建博客的日期和时间，这个也会在博客创建时自动生成；
  • tags 是文章的标签，可以像上面的示例一样另起一行，然后以短横线开头，中间隔一个空格，然后书写想要给文章加的标签，可以给博客添加多个标签，只需要另起一行即可；
  • categories 是文章的分类，书写方法同上面的 tags；
  • index_img 是文章在首页展示的封面，后面需要填写的参数是图片的 url 链接，具体也可以看上面的例子；
  • banner_img 是文章详细页面展示的页首图，写法同上面的 index_img，这里为了方便，其参数值就和上面的 index_img 保持一致（实际写博客的过程中也建议保持一致，这样比较省事儿）。
